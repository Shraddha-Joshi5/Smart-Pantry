# Smart-Pantry

This is a **Streamlit web app** that allows users to upload an **image** or a **video**, and detects objects in that using **Ultralytics YOLOv8**

## Features

- Upload files (`mp4`, `mov`, `avi`, `jpg`, `jpeg`, `png`) directly from your computer of your refrigerator/ your pantry.   
- Detects objects in the file using **YOLOv8**.  
- Displays the image/ frame with bounding boxes and object labels of the food items detected.  
- Shows detected object names below the frame.

## Libraries Needed
- streamlit
- opencv-python
- pillow
- ultralytics
- matplotlib

\n

If you want to run the app: \n
download the `app.py` file and execute `streamlit run app.py` in your terminal
